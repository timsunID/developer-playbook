# Developer Playbook

Personal documentation site for repeatable software setup workflows.

## Purpose

This repository stores practical, step-by-step processes for:

- fresh machine setup
- core tool installation
- project bootstrap workflows
- working standards
- troubleshooting runbooks
- reusable documentation templates

## Tech

- [Docusaurus](https://docusaurus.io/) (classic preset, TypeScript)
- pnpm

## Local development

```bash
pnpm install
pnpm start
```

## Build

```bash
pnpm build
```

## Structure

Primary content lives under `docs/` and is organized by setup area:

- `getting-started`
- `machine-setup`
- `project-setup`
- `workflows`
- `standards`
- `troubleshooting`
- `templates`
