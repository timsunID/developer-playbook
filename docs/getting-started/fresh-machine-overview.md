# Fresh Machine Overview

## Purpose

Provide a single path to set up a new development machine from zero.

## When to use it

- New laptop or desktop setup
- Clean operating system install
- Rebuilding your development environment after a reset

## Prerequisites

- Admin access on the machine
- Internet connection
- Access to your GitHub account

## Steps

1. Complete [Machine Setup](../machine-setup/README.md) in order.
2. Run [Verify Machine Setup](../machine-setup/verify-machine-setup.md).
3. Complete [Project Setup](../project-setup/README.md) for your first project.
4. Save any machine-specific notes under [Troubleshooting](../troubleshooting/README.md).

## Verify it works

- Git, Node.js, and pnpm are available in a new terminal session.
- VS Code opens projects correctly.
- SSH authentication to GitHub succeeds.

## Common mistakes

- Skipping shell restart after installing tooling
- Installing Node directly instead of through fnm
- Forgetting to enable Corepack before using pnpm

## Notes / best practices

- Follow the steps in order.
- Keep this page short and link to detailed process pages.
