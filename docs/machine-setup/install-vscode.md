# Install VS Code

## Purpose

Install Visual Studio Code as the primary editor for development work.

## When to use it

- Fresh machine setup
- Editor reinstall

## Prerequisites

- Internet connection

## Steps

1. Install VS Code using your platform package manager or installer.
2. Sign in to sync settings (optional).
3. Install baseline extensions required for your workflows.

## Verify it works

- `code --version` returns a version.
- `code .` opens the current project directory.

## Common mistakes

- PATH integration not enabled for `code` command
- Restoring too many extensions before base setup is stable

## Notes / best practices

- Keep a small baseline extension set.
- Capture extension standards in this playbook as needed.
